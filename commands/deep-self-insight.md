# 深层自我洞察命令

## 模块目标

基于长期阅读记录、划线、笔记和书评，分析用户的心路变化、核心追求、前后矛盾、核心焦虑、思维盲区，以及逃避与渴望。

> 注意：本模块只用于自我反思，不做心理诊断。所有结论都必须基于阅读证据，并使用谨慎表达。

## 命令列表

```text
/mind-journey
/core-pursuit
/ten-year-answer
/contradiction-check
/core-anxiety
/blind-spots
/avoid-and-desire
```

## `/mind-journey`

### 目标

分析用户长期阅读中的心路变化。

### 输出

```markdown
# 心路变化历程

## 1. 早期关注点
## 2. 中期转向
## 3. 近期主题
## 4. 长期主线
## 5. 关键转折点
## 6. 当前阶段判断
## 7. 下一阶段可能会追问的问题
```

## `/core-pursuit`

### 目标

分析用户真正追求的是什么。

### 输出

```markdown
# 核心追求分析

## 1. 表层追求
## 2. 深层追求
## 3. 这个追求想解决什么问题
## 4. 它带来的代价
## 5. 当前最清醒的回应方式
```

## `/ten-year-answer`

### 目标

推演用户当前反复追问的问题，十年后可能会变成什么答案。

### 输出

```markdown
# 十年后的答案

## 1. 你现在真正追问的问题
## 2. 你现在倾向的答案
## 3. 这个问题未来可能如何升级
## 4. 十年后更成熟的答案
## 5. 现在可以提前布局什么
```

## `/contradiction-check`

### 目标

找出用户笔记和划线中的前后矛盾、价值冲突和认知变化。

### 可识别矛盾

- 自由 vs 安全
- 长期主义 vs 短期反馈
- 学习 vs 行动
- 表达欲 vs 暴露恐惧
- 系统控制 vs 接纳不确定
- 独立创造 vs 被认可

### 输出

```markdown
# 前后矛盾检测

## 1. 主要矛盾列表
## 2. 每组矛盾的证据
## 3. 它可能代表的认知变化
## 4. 哪些矛盾是成长信号
## 5. 当前最需要处理的一组矛盾
```

## `/core-anxiety`

### 目标

分析字里行间隐藏较深的核心焦虑。

### 输出

```markdown
# 核心焦虑分析

## 1. 表层焦虑
## 2. 深层焦虑
## 3. 触发来源
## 4. 可能的防御方式
## 5. 更建设性的面对方式
```

## `/blind-spots`

### 目标

分析思维盲区和认知偏见。

### 可识别盲区

- 工具理性偏重
- 知识替代行动
- 技术乐观偏见
- 成功叙事偏见
- 情绪成本低估
- 过度自我要求
- 低估复利周期

### 输出

```markdown
# 思维盲区与认知偏见

## 1. 可能的思维盲区
## 2. 支撑证据
## 3. 它如何影响选择
## 4. 如何用行动验证或修正
## 5. 推荐补充阅读
```

## `/avoid-and-desire`

### 目标

分析用户可能在逃避什么，又在渴望什么。

### 输出

```markdown
# 逃避与渴望

## 1. 你可能正在逃避什么
## 2. 逃避背后的恐惧
## 3. 你可能正在渴望什么
## 4. 渴望背后的价值
## 5. 下一步最小行动
```

## 质量要求

- 必须写“可能”“倾向于”“从笔记看”。
- 必须引用阅读证据。
- 不要做人格定论。
- 不要做医学或心理诊断。
- 每个洞察都必须给出一个可执行下一步。
