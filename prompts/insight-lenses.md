# 多视角洞察提示词模板

## 使用方式

在 `/weread-loop` 中，根据用户选择的 `--lens` 执行对应洞察组合。

每个洞察都应该输出：

```markdown
### 洞察名称

**一句话洞察**：...

**阅读证据**：
- ...
- ...

**解释**：...

**可能盲区**：...

**下一步行动**：...
```

## Default 默认洞察组合

### 1. 全局扫描 / `global-scan`

**目标**：从全部阅读记录里发现隐藏主线。

提示词：

```text
请基于用户的书架、阅读记录、划线、笔记和书评，做一次全局扫描。
不要只总结最近读了什么，而是找出这些内容背后反复出现的长期主题、问题意识和认知迁移。
请回答：
1. 用户最近最关注什么？
2. 哪些主题反复出现？
3. 哪些问题可能已经持续很久？
4. 用户的阅读正在从什么转向什么？
5. 当前最值得保留的一句话洞察是什么？
```

### 2. 动力雷达 / `meaning-radar`

**目标**：分析用户真正被什么驱动。

```text
请从用户的划线和笔记中分析其意义来源和动力核心。
重点识别：哪些内容让用户反复停留、划线、记录想法？这些内容背后代表什么价值？
请区分表层兴趣和深层动力。
输出：动力来源、意义缺口、最核心驱动力、一个反思问题。
```

### 3. 当前张力 / `core-conflict`

**目标**：找出用户当前最核心的矛盾。

```text
请找出用户笔记和划线中最明显的内在张力。
例如：自由 vs 安全、长期主义 vs 即时反馈、学习 vs 行动、表达欲 vs 暴露恐惧、系统控制 vs 接纳不确定。
请说明这组张力的证据、影响和下一步处理建议。
```

### 4. 价值澄清 / `value-clarification`

**目标**：看清用户真正重视什么。

```text
请从用户笔记中提炼其真正重视的价值。
不要只看用户明确说出来的目标，而要结合反复划线、反复评论、长期关注的主题推断。
请输出：表层目标、深层价值、价值冲突、优先级建议。
```

### 5. 盲区探索 / `blindspot-explore`

**目标**：找出用户看不见但可能重要的问题。

```text
请从用户阅读偏好、笔记表达和缺失主题中，分析可能的思维盲区。
可以关注：工具理性偏重、知识替代行动、技术乐观偏见、成功叙事偏见、情绪成本低估、过度自我要求。
请谨慎表达，不要下诊断式结论。
```

### 6. 他者视角 / `third-person-view`

**目标**：像朋友一样观察用户。

```text
请用一个了解用户的朋友视角，温和但诚实地描述用户在这些阅读记录中呈现出来的状态。
请回答：他最近像是在为什么努力？他可能忽略了什么？他最值得被肯定的地方是什么？他下一步最该面对什么？
```

### 7. 个人飞轮 / `personal-flywheel`

**目标**：找出优势、积累和正反馈。

```text
请从用户的长期阅读主题、笔记和输出线索中，构建一个个人复利飞轮。
包括：核心需求、已有优势、可积累资产、正反馈机制、阻塞点、下一步启动动作。
```

### 8. 反常识洞见 / `contrarian-insights`

**目标**：提炼不寻常、有传播价值的观点。

```text
请从用户近期阅读和笔记中找出 3-5 个反常识洞见。
这些洞见应该有内容创作价值，不要是普通鸡汤。
每个洞见需要包含：标题、解释、来源证据、适合转成什么内容。
```

## Deep 深度自我洞察组合

### 心路变化 / `mind-journey`

```text
请根据用户长期笔记和划线，按时间线分析其心路变化。
输出：早期关注点、中期转向、近期主题、长期主线、关键转折点、当前阶段判断。
```

### 核心追求 / `core-pursuit`

```text
请分析用户真正追求的是什么。
区分表层追求和深层追求，并说明这个追求想解决什么问题、会带来什么代价、现在应该如何更清醒地面对。
```

### 十年答案 / `ten-year-answer`

```text
请推演用户当前反复追问的问题，如果再过十年，答案可能会如何变化。
输出：当前问题、当前答案、未来变化、十年后答案、现在可以提前布局的事。
```

### 核心焦虑 / `core-anxiety`

```text
请从用户字里行间分析其可能隐藏最深的核心焦虑。
必须谨慎表达，只能说“可能”“倾向于”“从笔记看”。
输出：表层焦虑、深层焦虑、触发来源、防御方式、温和建议。
```

### 逃避与渴望 / `avoid-and-desire`

```text
请分析用户可能正在逃避什么、又在渴望什么。
输出：正在逃避的事、正在渴望的事、逃避背后的恐惧、渴望背后的价值、下一步小行动。
```

## Decision 决策洞察组合

### 反向推演 / `inversion-thinking`

```text
请用逆向思考分析用户当前目标。
先回答：如果这件事失败，最可能因为什么？应该避免哪些愚蠢错误？哪些假设最危险？
最后给出避免失败的行动清单。
```

### 二阶思考 / `second-order-thinking`

```text
请分析用户当前选择或主题的一阶后果、二阶后果和长期影响。
尤其要指出短期舒服但长期有害、短期困难但长期有益的路径。
```

### 芒格视角 / `munger-view`

```text
请用查理·芒格式的多元思维模型分析用户的阅读和决策问题。
可使用：逆向思维、机会成本、激励机制、安全边际、误判心理学、能力圈等。
```

### 塞涅卡视角 / `seneca-view`

```text
请用斯多葛视角分析用户的焦虑和行动。
区分：可控、不可控、部分可控。
输出：应该放下什么、应该行动什么、如何减少内耗。
```

## Content 内容创作洞察组合

### 主题生长 / `theme-extension`

```text
请从用户笔记中找出可以继续生长的内容主题。
每个主题给出：主题名称、已有素材、可延展问题、适合平台、建议标题。
```

### 影响力地图 / `influence-map`

```text
请找出用户阅读中反复出现的作者、人物、理论来源和思想影响。
分析这些人物如何影响用户的问题意识和内容方向。
```

### 小红书内容 / `book-to-xhs`

```text
请把用户的一组读书笔记转成小红书图文内容。
输出：标题、正文、标签、每张图的内容建议。
```

### 短视频口播 / `book-to-video-script`

```text
请把用户的一组读书笔记转成短视频口播。
输出：30 秒、90 秒、3 分钟三个版本，并给出分镜建议。
```

### 卡片内容 / `book-to-cards`

```text
请把读书笔记转成知识卡片。
输出：封面卡、观点卡、金句卡、行动卡、总结卡。
```
